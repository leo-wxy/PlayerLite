## ADDED Requirements

### Requirement: 播放展开页前景层级遵循共享主题 token
系统 SHALL 在保留沉浸式封面背景与现有播放行为的前提下，让播放页的主操作、激活态、辅助信息与弱化容器遵循共享主题 token，而不是继续混用彼此割裂的强调色和通用灰底。

#### Scenario: 主操作与激活态使用统一主强调语义
- **WHEN** 用户查看播放页中的主操作按钮、当前激活状态或关键入口
- **THEN** 系统使用共享主题中的主强调语义突出这些元素
- **AND** 相同语义的按钮和文本在播放页与其他音乐页面之间保持一致

#### Scenario: 辅助信息和次级容器保持克制层级
- **WHEN** 用户查看播放页中的歌手信息、歌词摘要、次级按钮或辅助状态
- **THEN** 系统使用次级信息色与克制的容器底色表达层级
- **AND** 不以过多高饱和色同时竞争页面注意力
