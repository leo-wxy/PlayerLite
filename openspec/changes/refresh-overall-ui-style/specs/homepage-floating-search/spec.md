## ADDED Requirements

### Requirement: 首页顶部搜索入口采用轻量圆角搜索框
系统 SHALL 在首页顶部将搜索入口展示为浅底、大圆角、低视觉重量的搜索框，并与独立搜索页使用一致的主题语言，而不是继续依赖偏重的通用搜索容器。

#### Scenario: 首页搜索入口展示图标与默认关键词
- **WHEN** 用户查看首页顶部搜索入口
- **THEN** 左侧展示搜索 icon
- **AND** 中间展示默认搜索关键词或占位文案
- **AND** 搜索框文字颜色与 icon 颜色遵循共享主题中的次级文本与主强调分工

#### Scenario: 搜索入口与首页内容保持统一宽度和安全区节奏
- **WHEN** 用户查看首页首屏
- **THEN** 搜索入口在顶部安全区下方保持稳定水平留白
- **AND** 它与 banner、推荐区块共享一致的内容宽度基线
- **AND** 不因为关键词轮播而导致搜索框整体抖动
