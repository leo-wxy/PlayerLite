## Reference Assets

这个目录用于存放 `refresh-overall-ui-style` 的本地样式参考图，方便后续继续对照 UI 细节。

约定：

- 实际图片文件放在 `references/assets/`
- 图片文件默认只作本地参考，不进入 Git 提交
- 若后续需要共享可提交的设计说明，请把文字结论写回 `proposal.md`、`design.md`、`specs/` 或 `tasks.md`

建议文件名：

- `assets/playlist-sheet-reference.png`
  播放列表 bottom sheet / Next Up 参考图
- `assets/search-empty-reference-v1.png`
  搜索页空查询态参考图（无返回键版本）
- `assets/search-empty-reference-v2.png`
  搜索页空查询态参考图（含返回键版本）
- `assets/search-results-reference.png`
  搜索结果页参考图
- `assets/home-overview-reference.png`
  首页发现流与 minibar 参考图

如果后续出现新的迭代图，直接按同一命名风格继续追加即可。
