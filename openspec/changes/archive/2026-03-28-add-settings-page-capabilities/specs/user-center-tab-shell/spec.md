## ADDED Requirements

### Requirement: 用户中心页提供设置页导航入口
系统 SHALL 在用户中心页的头部资料区提供稳定可见的设置页入口，用于打开独立设置页，而不是把设置能力继续散落在快捷入口区或主内容区。

#### Scenario: 用户中心页展示设置入口
- **WHEN** 用户进入用户中心页面
- **THEN** 系统在头部资料区展示设置页入口
- **AND** 该入口在已登录态和游客态都保持可见
- **AND** 现有“喜欢”“最近”“本地”等快捷入口继续保留在原有快捷入口区

#### Scenario: 点击设置入口打开设置页
- **WHEN** 用户在用户中心页点击设置页入口
- **THEN** 系统打开独立设置页
- **AND** 当前主壳层选中的一级 Tab 仍保持为用户中心
