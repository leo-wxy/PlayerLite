## ADDED Requirements

### Requirement: 设置页作为标准主动退出登录入口
系统 SHALL 让设置页成为在线账户的标准主动退出登录入口，并复用现有用户会话清理与前台状态同步语义；退出登录 MUST 不影响本地播放能力和其他不受登录态约束的入口可用性。

#### Scenario: 已登录用户从设置页退出登录
- **WHEN** 系统已恢复有效用户会话且用户在设置页确认退出登录
- **THEN** 系统清理本地持久化的用户会话与用户资料快照
- **AND** 设置页与用户中心同步降级为未登录状态
- **AND** 后续受登录态约束的在线入口重新要求用户登录
- **AND** 本地播放能力继续可用

### Requirement: 设置页对登录态变化保持同步
系统 SHALL 让设置页与现有登录页、用户中心页共享同一套用户会话状态；当登录态失效或重新建立时，设置页中的账户分组 MUST 同步更新。

#### Scenario: 设置页可见时会话失效
- **WHEN** 用户正在查看设置页且系统检测到当前会话失效
- **THEN** 设置页账户分组同步降级为未登录状态
- **AND** 页面不再展示退出登录操作
