## ADDED Requirements

### Requirement: 用户中心页提供个人主页快捷入口区
系统 SHALL 在用户中心页的头部资料区下方提供一组可点击的快捷入口，用于承载个人页的高频操作与分区跳转；快捷入口以“图标 + 文案标签”的形式展示，并与账户域视觉语言保持一致。

#### Scenario: 用户中心展示快捷入口区
- **WHEN** 用户进入用户中心页面
- **THEN** 系统在头部资料区下方展示快捷入口区
- **AND** 快捷入口区至少包含“喜欢”“最近”“本地”三个入口
- **AND** 快捷入口区在游客态仍保持可见

#### Scenario: 点击“喜欢”快捷入口打开“喜欢”子页面
- **WHEN** 用户在用户中心页点击“喜欢”快捷入口
- **THEN** 系统打开“喜欢”子页面
- **AND** 子页面以 Tab 形式承载不同类型的“喜欢/收藏”内容分区

#### Scenario: “喜欢”子页面提供 Tab 切换展示收藏内容
- **WHEN** 用户进入“喜欢”子页面
- **THEN** 系统展示至少 3 个 Tab，用于切换展示不同类型的收藏内容
- **AND** Tab 至少包含“歌手”“MV”“专栏”之一（可按接口能力扩展）

#### Scenario: “喜欢”子页面在游客态展示登录引导
- **WHEN** 用户在游客态进入“喜欢”子页面
- **THEN** 系统展示登录引导，而不是直接请求在线收藏数据

#### Scenario: “喜欢”子页面点击未支持落地时给出提示
- **WHEN** 用户点击“喜欢”子页面任一列表 item
- **AND** 当前版本无法对该 item 执行跳转或播放落地
- **THEN** 系统给出明确提示，而不是静默无响应

#### Scenario: 点击“最近”快捷入口打开最近播放子页面
- **WHEN** 用户在用户中心页点击“最近”快捷入口
- **THEN** 系统打开最近播放子页面

#### Scenario: 点击“本地”快捷入口进入本地歌曲页面
- **WHEN** 用户在用户中心页点击“本地”快捷入口
- **THEN** 系统打开本地歌曲页面

### Requirement: 最近播放子页面展示最近播放歌曲
系统 SHALL 提供最近播放子页面，用于展示当前账号最近播放过的歌曲列表，并提供加载、空态、错误与重试反馈；该子页面 MUST 只在已登录态下请求在线数据。

#### Scenario: 已登录用户进入最近播放子页面后加载最近播放歌曲
- **WHEN** 用户已登录并进入最近播放子页面
- **THEN** 系统发起最近播放歌曲加载并展示加载态
- **AND** 当加载成功且存在结果时，系统展示最近播放歌曲列表
- **AND** 每个列表 item 至少展示歌曲标题与歌手信息

#### Scenario: 最近播放歌曲为空时展示空状态
- **WHEN** 用户已登录并进入最近播放子页面
- **AND** 最近播放歌曲请求成功但返回空结果
- **THEN** 系统展示空状态提示

#### Scenario: 最近播放歌曲加载失败时展示重试
- **WHEN** 用户已登录并进入最近播放子页面
- **AND** 最近播放歌曲请求失败且不属于会话失效
- **THEN** 系统展示错误提示与重试入口

#### Scenario: 游客态进入最近播放子页面时展示登录引导
- **WHEN** 用户在游客态进入最近播放子页面
- **THEN** 系统展示登录引导而不是直接请求最近播放数据

#### Scenario: 点击最近播放歌曲缺少落地时给出提示
- **WHEN** 用户点击最近播放歌曲列表中的任一 item
- **AND** 当前版本无法对该 item 执行跳转或播放落地
- **THEN** 系统给出明确提示，而不是静默无响应

## MODIFIED Requirements

### Requirement: 用户中心已登录内容区通过 Tab 切换展示个人收藏内容
系统 SHALL 在用户中心的已登录内容区展示当前用户自己创建的歌单列表，并以个人主页式结构承载头部资料区、快捷入口区与歌单列表区；首版不要求在个人主页中继续通过 Tab 展开展示收藏歌手与收藏专栏等扩展内容。

#### Scenario: 已登录用户进入用户中心时看到用户自己的歌单列表
- **WHEN** 用户已登录并进入用户中心页面
- **THEN** 系统在头部资料区与快捷入口区下方展示“创建歌单”列表区
- **AND** 列表区展示当前用户自己创建的歌单

#### Scenario: 用户自己的歌单为空时展示空状态
- **WHEN** 用户已登录并进入用户中心页面
- **AND** 当前用户没有创建的歌单
- **THEN** 系统在歌单列表区展示空状态提示

#### Scenario: 用户自己的歌单加载失败时展示局部重试
- **WHEN** 用户已登录并进入用户中心页面
- **AND** 用户歌单请求失败且不属于会话失效
- **THEN** 系统在歌单列表区展示错误提示与重试入口
- **AND** 头部资料区与快捷入口区继续保持可用

#### Scenario: 点击用户歌单进入歌单详情页
- **WHEN** 用户点击歌单列表区中的任一歌单 item
- **THEN** 系统为该 item 提供明确点击反馈
- **AND** 点击后进入对应歌单详情页
