## ADDED Requirements

### Requirement: 用户中心受登录态约束的内容请求复用当前会话鉴权
系统 SHALL 让用户中心中的收藏歌手、收藏的专栏和用户歌单请求复用当前有效用户会话对应的认证请求头；这些请求 MUST 沿用现有受限在线接口的鉴权方式，并至少携带当前会话中的 `Cookie`，同时在会话失效时遵守统一的登录态降级规则。

#### Scenario: 已登录用户请求个人中心受限内容时沿用当前 Cookie
- **WHEN** 系统已恢复有效用户会话，且用户中心发起收藏歌手、收藏专栏或用户歌单请求
- **THEN** 系统沿用当前会话生成的认证请求头发起请求
- **AND** 请求至少携带当前会话中的 `Cookie`

#### Scenario: 未登录用户进入用户中心时不请求受限内容
- **WHEN** 当前没有有效用户会话，且用户进入用户中心页面
- **THEN** 系统不发起收藏歌手、收藏专栏或用户歌单等受登录态约束的内容请求
- **AND** 系统保持未登录引导与基础页面承载可用

#### Scenario: 用户中心受限内容接口返回未登录时统一降级
- **WHEN** 收藏歌手、收藏专栏或用户歌单接口返回明确的未登录、会话失效或认证过期结果
- **THEN** 系统清理失效的用户会话并将当前用户状态统一降级为未登录
- **AND** 用户中心页面同步回到未登录引导状态
