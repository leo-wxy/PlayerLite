## MODIFIED Requirements

### Requirement: 个人中心页提供可直接进入的本地歌曲入口
系统 MUST 在个人中心页提供一个稳定可见的“本地歌曲”入口，用于进入独立的本地歌曲页面；该入口不应依赖登录状态。

#### Scenario: 登录态个人中心页展示本地歌曲入口
- **WHEN** 用户进入登录态个人中心页
- **THEN** 系统展示“本地歌曲”入口
- **AND** 该入口不会替代现有内容面板和账号动作

#### Scenario: 游客态个人中心页展示本地歌曲入口
- **WHEN** 用户进入游客态个人中心页
- **THEN** 系统仍展示“本地歌曲”入口
- **AND** 该入口与登录按钮并存
