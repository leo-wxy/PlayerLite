## ADDED Requirements

### Requirement: 歌手详情页热门歌曲支持整体播放
系统 SHALL 在歌手详情页热门歌曲区域提供整体播放入口，并以当前歌手详情页已加载的热门歌曲列表作为新的播放上下文。

#### Scenario: 点击全部播放后以热门歌曲集合替换当前队列
- **WHEN** 用户在歌手详情页点击“全部播放”
- **THEN** 系统使用当前歌手详情页已加载的热门歌曲集合作为新的当前播放列表
- **AND** 将首个有效曲目设为激活项并开始播放
- **AND** 在播放列表替换成功后直接打开播放器展开页

#### Scenario: 点击热门歌曲后以当前歌手上下文指定播放
- **WHEN** 用户在歌手详情页点击某一首热门歌曲
- **THEN** 系统使用当前歌手详情页已加载的热门歌曲集合作为新的当前播放列表
- **AND** 将被点击歌曲的 index 设为激活项并开始播放
- **AND** 在激活项切换成功后直接打开播放器展开页
